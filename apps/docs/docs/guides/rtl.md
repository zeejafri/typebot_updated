# Enable RTL

- Head over to the "Theme" tab
- Paste the following Custom CSS:

```css
.typebot-container {
  direction: rtl;
}

.typebot-avatar-container {
  margin-right: 0;
  margin-left: 0.5rem;
}

.guest-container {
  margin-left: 0;
  margin-right: 50px;
}
```

If there is any visual flaw, [contact me](mailto:support@typebot.freshdesk.com) so that I can adjust the CSS rules
